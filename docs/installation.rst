============
Installation
============

.. The following installation instructions are recommended if the plugin is uploaded to pypi.

Stable release
--------------

To install pysteps-importer-metranet, run this command in your terminal:

.. code-block:: console

    $ pip install pysteps_importer_metranet


.. The following installation instructions are recommended if the plugin needs to be installed
   from its sources (e.g. directly from the github repo).

From sources
------------

The latest development version of pysteps-importer-metranet can be installed using
pip by running in a terminal::

    pip install git+https://github.com/YOUR_GITHUB_USERNAME/pysteps-importer-metranet

Or, from a local copy of the repo::

    git clone https://github.com/YOUR_GITHUB_USERNAME/pysteps-importer-metranet
    cd pysteps-importer-metranet
    pip install .

The above commands install the latest version of the **master** branch,
which is continuously under development.
